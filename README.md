# MyFormula
A local-first, single-file HTML app that lets you:
- define variable names
- define formulas using those variables
- enter values and instantly evaluate results

# Features
Fully editable
-add, edit, or delete variables, formulas, and calculators
-reoder variables
-built in and custom units
-import/export calculators: makes it easy to share caclulators with others as well as using AI to generate new calculators for you.
    To use AI, simply export a random list of calculators, tap copy, paste to your ai agent as a template on how the json file should be set up, then explain the calculator or converter you want, AI gives you a json file you can copy then paste to import calculators in the app
-the help menu explains all supported math and function as well as examples
